# Pokemon Search Engine

A full stack Pokemon search application built using Spring Boot and React.

## Features

- Search Pokemon by name
- Display Pokemon details
- REST API integration
- Caching support
- Error handling
- Responsive UI

## Tech Stack

### Backend
- Java
- Spring Boot

### Frontend
- React
- Axios

## Run Backend

```bash
mvn spring-boot:run
```

## Run Frontend

```bash
npm install
npm run dev
```

## API Endpoint

```text
GET /pokemon/{name}
```
